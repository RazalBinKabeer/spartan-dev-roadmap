Chicken Biriyani

Ingredients required

  • Chicken
  • Rice
  • Masala
  • Oil
  • Ghee
  • Tomato
  • Onion
  • Cashew
  • Corriander leaves
  • Mint leaves

Preparation

  1. Boil Water
  2. Boil Rice
  3. Add Masala
  4. Add Chicken
  5. Add Oil
  6. Add Ghee
  7. Add Tomato
  8. Add Onion
  9. Add Cashew
  10. Add Corriander leaves
  11. Add Mint leaves
  12. Boil for 20 minutes

Chicken Biriyani ready
